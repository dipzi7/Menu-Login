Siapa Kami?

Kami Adalah Siswa Smk Hikma Yapis Yang Di Tugaskan Untuk Membuat Website Ini

Website ini dibuat sebagai sarana untuk menampilkan bio data siswa dengan tampilan yang sederhana namun tetap rapi. Di dalamnya, setiap informasi penting seperti nama, identitas kelompok, dan data tambahan disusun secara terstruktur agar mudah dipahami. Dengan adanya website ini, penyajian data tidak lagi hanya berbentuk tulisan biasa, tetapi menjadi lebih menarik dan interaktif.

Selain sebagai media informasi, website ini juga dapat dijadikan contoh dalam penerapan dasar-dasar pemrograman web. Penggunaan HTML dan CSS membantu memperkenalkan konsep desain web yang praktis, sehingga bisa menjadi bahan pembelajaran sekaligus portofolio bagi pengembangnya. Harapannya, website ini dapat bermanfaat baik dalam konteks akademik maupun pengembangan keterampilan digital.
